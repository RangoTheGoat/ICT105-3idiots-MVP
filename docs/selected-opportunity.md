# Selected Semester Project Opportunity

## Selected Project Title
Campus Lost and Found Reporting System

## Target User
University students who lose or find personal belongings.

## Problem Statement
Students often lose personal items on campus and have no centralized platform to report or search for them.

## Current Alternative
Students currently use chat groups, ask friends, or contact the security office.

## Proposed IT Solution
A web-based platform that allows students to report lost items, upload found items, and search existing reports.

## Why This Opportunity Was Selected
The project achieved the highest NUF score. It solves a real problem, is useful for students, and can be developed within one semester.

## Why This Is Feasible Without IoT or Advanced Cybersecurity
The system only requires a web interface, database, and search functionality. No hardware devices or advanced security implementation are needed.

## Possible MVP Type
HTML/CSS/JS Prototype

## Initial Validation Plan for Lab 03
Interview university students and collect evidence regarding how frequently they lose items and how they currently search for them.
